Connection Speed Tester

== Description ==

This widget makes it possible that any visitor of your blog can test his bandwidth speed. The widget ist hosted on Amazon S3 storage and checks the speed from the nearest server worldwide.

It is multilingual. (English and German)

== Installation ==

To use the widget you only have to do this:

1. Upload `speedtest-widget.php` to the `/wp-content/plugins/` directory
2. Activate the widget through the 'Plugins' menu in WordPress
3. Configure the widget through the 'Options' menu in WordPress
4. You have to copy the embed code into the insert-field and save changes!
5. Include the widet throught the 'Design-Widgets' menu in WordPress


== Frequently Asked Questions ==

= Which languages are supported? =

The widget will be displayed in the user's language. Following languages are supported:

*		English
*		German
